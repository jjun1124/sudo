강력한 핵심 기능

Web Unlocker

복사 방지, 우클릭 제한이 걸린 웹사이트에서도 자유롭게 콘텐츠를 선택하고 복사할 수 있습니다. stopPropagation 기술로 이벤트 차단을 무력화합니다.

Smart Dark Mode

모든 웹사이트를 즉시 다크 모드로 전환합니다. CSS 인버트 필터와 색상 보정을 통해 눈의 피로를 최소화하며, 이미지와 영상은 원래 색상을 유지합니다.

sudo_note

브라우징 도중 떠오른 아이디어를 즉시 기록하세요. 텍스트 영역에 입력된 내용은 익스텐션을 닫아도 안전하게 저장됩니다.

Work Timer

작업 시간을 정밀하게 측정하세요. 백그라운드 서비스 워커가 익스텐션 팝업이 닫혀 있는 동안에도 시간을 정확하게 계산합니다.

직관적인 인터페이스

개발자 지향적인 다크 테마 디자인으로 구성되었습니다. 복잡한 설정 없이 스위치 하나로 기능을 켜고 끌 수 있습니다.

  • Chrome Manifest V3 표준 준수
  • 초경량 서비스 워커 아키텍처
  • 실시간 스토리지 동기화
